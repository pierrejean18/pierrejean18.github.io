Projet Web Scraping Tanks

Ce projet consiste à extraire des informations sur des tanks à partir d'une page web, en utilisant Python et les bibliothèques pandas, requests et BeautifulSoup.

Code

import pandas as pd
import requests
from bs4 import BeautifulSoup
import re
import csv

URL ="https://www.oryxspioenkop.com/2022/02/attack-on-europe-documenting-equipment.html"
response = requests.get(URL)
# ...

Explication

Le script commence par récupérer le contenu de la page, puis utilise BeautifulSoup pour analyser le HTML et extraire les informations sur les tanks. Il sauvegarde les résultats dans un fichier CSV.

Résultats

Les données extraites incluent les modèles de tanks, leur état, et des liens vers des images ou vidéos associées.
